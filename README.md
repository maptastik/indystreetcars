indystreetcars
==============

Repo for Indianapolis street car work

Work here inspired by Kevin Kastner [(Link to post)](http://www.urbanindy.com/2014/12/16/indys-streetcar-network-91-years-later/)

Kevin is responsible for creating the original street car network dataset
